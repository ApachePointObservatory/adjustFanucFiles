BuildForMac README

To build a Mac OS X application:

% python setup.py [--quiet] py2app

Notes:

  • Requires the following python packages
    • py2app, available from http://pythonmac.org/packages/.
    • RO Package
    • scipy
    • numpy
    • The fitPlugPlateMeas package from svn+ssh://sdss3svn@sdss3.org/repo/platemeas/fitPlugPlateMeas
  • I recommend that you manage RO and fitPlugPlateMeas using eups; alternatively you can put both packages on your python path or specify their paths in a file named extraPaths.txt in the BuildForMac directory.
  • If you install your own Python then it will be included in the package, which makes it more portable. See Python on the Macintosh for information.
